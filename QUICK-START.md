Your data is stored in **4 different places** for maximum safety:

1. **Supabase Cloud Database** (main storage)
   - Professional cloud hosting
   - Automatic daily backups
   - 99.9% uptime guarantee

   - Save to Google Drive, Dropbox, etc.

## ğŸ“¤ How to Backup Your Data (Recommended Weekly)

1. Go to **Settings** page (gear icon in sidebar)
2. Scroll to **Data Management** section
3. Click **"Export Data"** button

   - Google Drive

   - External USB drive

## ğŸ”„ How to Restore Data

### If something goes wrong:
1. Go to **Settings** page
2. Click **"Import Data"** button
3. Select your backup JSON file
4. Click Open
5. Done! All data restored

### If you see an error:
1. Click **"Recover from Backup"** button
2. System will load from local backup
3. Continue working
